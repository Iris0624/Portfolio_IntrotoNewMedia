Introduction to New Media


My Code and Digital Narrative

This project challenged me to create something within boundaries yet add my own unique touches to it. I included some photos that I took myself to show my interest in photography. As I made this webpage I made sure to test out each edit I made in the code carefully and ensure that it works properly. I did have trouble in making one function work, so I am going to continue experimenting until it works the way I want it to.

Webtext Experiment

For the webtext experiment I allowed myself to step out of my comfort zone and try create this webpage without structure. I usually like to plan out my projects and each step I will take to get to the end product but after reading one of Derrida's works, I was inspired to try something new. I played around with different pieces of code and discovered that I can make GIFS interactive. I was able to add little GIF stickers on the page and users can drag them around however they like. I also was able to present this in class which allowed me to have practice speaking in front of peers and put down my worries about whether or not my work was perfect. This project challenged my perfectionist mindset and taught me to be more carefree and let the creativity flow.

Day in NYC

This project was fun for me as I enjoyed the structure it had compared to the previous project. Even so, I still enjoy the creative freedom we had for our projects overall. I also was able to use my previous coding skills and that really helped me get through the project smoothly. I learned a lot when workshopping this project. I was inspired by my peers after seeing their creations and it made me reflect on what I could do to better my own. Me and my group of peers helped each other with issues we faced, especially the Github uploading portion. We gave each other feedback. This helped me learn what I can improve and things that I did well. Overall, It was a fun and enlightful experience.
